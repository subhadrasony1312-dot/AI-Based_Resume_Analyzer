# 🤖 AI-Based_Resume_Analyzer

📌 Overview
<br>
The AI-Based Resume Analyzer is a smart web application that helps recruiters and candidates evaluate how well a resume matches a job description. It uses Natural Language Processing (NLP) and Machine Learning (TF-IDF + Cosine Similarity) to provide accurate match scores and insights.
<br>
🚀 Live Demo :
<br>
👉 https://ai-based-resume-analyzer-wk17.onrender.com
<br>
✨ Features :
<br>
📄 Upload multiple resumes (PDF & DOCX)
<br>
🧠 AI-based resume vs job description matching
<br>
📊 Match score using TF-IDF & Cosine Similarity
<br>
🔍 Missing skills detection
<br>
📈 Visual graph for comparison
<br>
📥 Download results as CSV
<br>
💾 Stores analysis history in PostgreSQL database
<br>
⌛ View previous analysis records
<br>
🎨 Modern UI with custom styling
<br>
🛠️ Tech Stack
<br>
🔹 Frontend :

<br>

Python - Programming Language

<br>
